# 📚 Teste-ED — Simulado de Estrutura de Dados

Simulado interativo desenvolvido como material de estudo para a avaliação da disciplina de **Estrutura de Dados**, do curso de Análise e Desenvolvimento de Sistemas — Senac.

---

## 📋 Sobre o Projeto

O simulado foi construído em HTML/CSS/JS puro e cobre os principais temas cobrados na avaliação, combinando questões de múltipla escolha com questões dissertativas práticas de implementação em Java.

---

## 🧩 Conteúdos Abordados

**Algoritmos de Ordenação e Busca**
- Ordenação pelo método Bubble Sort (método bolha)
- Busca Linear (sequencial)
- Busca Binária — condições de aplicabilidade em arrays vs. listas ligadas

**Arrays e Alocação de Memória**
- Alocação dinâmica em tempo de execução (`new`)
- Funcionamento do Garbage Collector em Java
- Referências e perda de ponteiro para objetos na heap

**Listas Simplesmente Ligadas**
- Estrutura de nós e referências (`inicio`, `atual`, `proximo`)
- Iteração em lista encadeada com variável auxiliar do tipo `Elemento`
- Inserção e remoção de células
- Contagem de elementos com critério (ex: valores pares)
- Elegibilidade para coleta pelo GC após desreferenciamento

**Implementação Prática em Java**
- Busca linear em array de objetos com retorno de índice
- Método `pesquisar(int id)` com estrutura de classe `Array` e `Pessoa`
- Método `contarPares()` em lista encadeada
- Instruções de remoção de célula específica

---

## ⚙️ Como Usar

Basta abrir o arquivo `indexED.html` diretamente no navegador — nenhuma dependência ou servidor é necessário.

1. Responda as questões de múltipla escolha selecionando a alternativa desejada.
2. Preencha as questões dissertativas nas áreas de texto.
3. Clique em **"Finalizar e Validar Respostas"** para ver o gabarito e o feedback de cada questão.
